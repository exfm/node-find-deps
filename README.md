# find-deps

Little helper tool for printing the name and latest version of all modules your code needs.

## Install

    npm install -g find-deps

## Example

Running `find-deps` on find-deps:

    find-deps[master]/$ find-deps
    {
        "esprima": "1.0.0",
        "request": "2.11.4",
        "when": "1.5.2"
    }
